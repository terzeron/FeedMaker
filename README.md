FeedMaker
=========

Utility set of making feeds from websites

Requirements
------------
1. Python & modules
	1) bs4
2. Perl & modules
	1) Digest::MD5
	2) HTML::Parser
	3) HTML::TreeBuilder::XPath
	4) JSON
	5) Modern::Perl
	6) Scalar::Util
	7) Text::Levenshtein
	8) URI::Encode
	9) XML::RSS
	10) XML::Simple
